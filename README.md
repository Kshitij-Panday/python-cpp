# Basic Competitive Programming (Python)

This repository contains the Python programs covered in my **Basic Competitive Programming** class.
The goal of these exercises is to build **basic programming logic and problem-solving skills using Python**.

---

## What I Learned

* Python syntax and basic programming structure
* Taking user input and displaying output
* Type casting between different data types
* Conditional statements (`if`, `else`)
* Loops (`for`, `while`)
* Working with **lists, dictionaries, and strings**
* Basic mathematical and logical problem solving

---

## Repository Contents

The programs in this repository include:

* Swapping two numbers
* Height conversion (feet → inches → centimeters)
* Reverse a number
* Simple interest calculation
* Temperature conversion (Celsius ↔ Fahrenheit)
* Positive / Negative / Zero check
* Working day or weekend check
* Student marks and percentage calculation
* Maximum value among numbers
* List operations and sorting
* Dictionary operations
* String operations and slicing
* Vowel and consonant counting
* Type casting examples
